# News Classification

## Requisits

- Python 3.13
- Llibreries necessàries (vegeu `requirements.txt`)

## Instal·lació

1. Crear i activar un entorn virtual:
```bash
python3.13 -m venv venv
source venv/bin/activate  # Linux/Mac

venv\Scripts\activate  # Windows
```

2. Instal·lar les dependències:
```bash
pip install -r requirements.txt
```

3. Crear la carpeta `assets` (en cas que no exisetixi)

## Execució

Executar l'script principal:
```bash
python task.py
```

## Resultats

L'script generarà:
- Visualitzacions guardades a la carpeta `assets/`
- Resultats de l'anàlisi impresos per consola
